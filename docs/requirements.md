# 需求说明

## 目的

本文档定义 `Transwarp Service Insight` 一阶段 `SLA 预诊助手` 的业务需求、功能需求、非功能需求、数据需求、权限需求、审核需求、日志追踪需求，以及 MVP 与增强需求。

## 适用范围

适用于需求评审、Demo 范围确认、任务拆解和后续实现规划。当前文档不包含代码实现，不代表已接入真实系统或真实数据。

## 业务需求

- 支持内部员工在 SLA 提交前进行问题预诊断。
- 支持识别信息缺口，提升 SLA 提交质量。
- 支持基于问题描述的初步分类建议，区分基础类、重复类、开源社区类和可能需要正式 SLA 的问题。
- 支持相似案例、处理手册、Wiki、内部培训素材和使用手册检索。
- 支持模拟 AIOps 指标和模拟 SLA 复盘数据摘要。
- 支持生成可人工编辑的 SLA 工单草稿。
- 支持处理完成后的复盘报告草稿生成。
- 支持人工审核和知识库沉淀流程。

## 功能需求

### SLA 预诊

- 检查问题描述是否包含发生时间、影响范围、业务对象、产品组件、版本信息、现象和已尝试操作。
- 识别缺失字段并生成补充问题。
- 提示可能影响判断的风险。
- 输出是否建议提交 SLA 草稿的辅助判断，但不得自动提交。

### 问题分类

- 辅助识别跑批延迟、任务等待、资源瓶颈、依赖异常、数据同步失败、基础使用问题、重复已知问题、开源社区类问题等类型。
- 输出分类依据、置信度和需要人工确认的字段。
- 支持多个候选分类。

### RAG 检索

- 从模拟售后知识库中检索处理手册和经验条目。
- 从模拟 AIOps 中 SLA 复盘数据检索相似复盘线索。
- 从模拟 Wiki、模拟内部培训素材和模拟使用手册中检索参考资料。
- 从模拟历史 SLA 中检索相似案例。
- 输出引用来源，不得虚构命中结果。
- 对不适用案例进行提示。

### AIOps 摘要

- 支持读取模拟指标数据。
- 摘要异常时间段、趋势变化和关联对象。
- 输出需人工核验的数据点。
- 不得将模拟指标相关性描述为最终根因。

### SLA 草稿生成

- 生成标题、问题描述、影响范围、优先级建议和附件清单。
- 标注待确认字段。
- 明确草稿需人工审核。
- 不得自动提交正式 SLA。

### 复盘报告生成

- 生成事件摘要、时间线、影响分析、原因分析草稿、处置过程和改进建议。
- 在缺少人工确认根因时，不生成最终根因。
- 生成知识库沉淀候选条目。

### 知识沉淀

- 支持从已审核复盘中提取知识条目。
- 标注适用范围、来源、审核状态和更新时间。
- 必须由提交人和处理人审核确认后沉淀。

## 非功能需求

- 可追溯：每条建议应保留依据来源。
- 可审核：关键输出必须支持人工确认和修改。
- 可解释：分类、建议和摘要应说明理由。
- 可控：不得自动执行高风险操作。
- 可扩展：后续可接入更多 SLA 类型和知识库来源，但必须经过权限、脱敏和审计设计。
- 可审计：保留用户输入、模型输出、人工修改和审核记录。

## 数据需求

Demo 阶段使用以下模拟数据：

- 模拟售后知识库条目。
- 模拟 AIOps 中 SLA 复盘数据。
- 模拟 Wiki 知识条目。
- 模拟内部培训素材。
- 模拟使用手册。
- 模拟处理手册。
- 模拟历史 SLA。
- 模拟任务调度指标。
- 模拟队列资源指标。
- 模拟报表生成记录。

所有模拟数据必须标注为“模拟数据”，不得出现真实客户、真实业务系统、真实历史 SLA、真实知识库内容或真实接口返回。

## 权限需求

后续实现应至少区分：

- 提交人：创建和补充 SLA 信息。
- 处理人：查看建议、更新处理过程、确认解决方案。
- 审核人：审核 SLA 草稿、复盘报告和知识沉淀内容。
- 管理员：维护知识库、标签和权限。

## 审核需求

必须人工审核的内容包括：

- SLA 是否正式提交。
- SLA 标题、影响范围和优先级。
- 问题分类。
- 处理建议是否采纳。
- 根因分析。
- 复盘报告。
- 知识库沉淀条目。

## 日志追踪需求

建议记录：

- 用户输入时间和内容摘要。
- Agent 输出和版本。
- 检索命中的知识来源。
- 模拟 AIOps 数据版本。
- 人工修改记录。
- 审核人、审核时间和审核意见。

## MVP 需求

- 完成标准文档体系。
- 确认总项目命名和一阶段 `SLA 预诊助手` 路线。
- 定义 Demo 场景和模拟数据结构。
- 定义 Agent 职责和边界。
- 定义 SLA 草稿和复盘模板。
- 定义任务拆解和进度管理方式。

## 增强需求

- 接入脱敏测试知识库。
- 支持多轮预诊问答。
- 支持知识库版本管理。
- 支持与工单系统的受控集成。
- 支持建议采纳率、预诊补全率、有效 SLA 处理效率和复盘沉淀率统计。

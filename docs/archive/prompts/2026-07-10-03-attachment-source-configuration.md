# 模拟附件、开放知识回答与知识来源配置升级提示词

## 使用时间

2026-07-10

## 使用目的

升级第一阶段纯前端 `SLA 智能预诊助手` Demo，增加管理员知识来源配置、严格来源隔离、开源资料降级、内置模拟附件和结构化辅助回答，并同步后续目标设计文档。

## 适用范围

适用于 `prototypes/sla-precheck-demo.html`、相关 PRD/交互/Mock/RAG/LLM/安全/API/观测/验收文档、提示词归档和 CHANGELOG。仅允许纯前端静态实现与设计草案，不生成后端代码，不调用真实接口、RAG、模型、互联网或内部资料。

## 提示词正文

在现有对话式预诊 Demo 中实现知识增强回答总开关、Wiki、历史 SLA、产品手册、开源资料和模型通用知识降级开关，默认全部开启，使用 `localStorage` 模拟管理员全局配置。每次预诊冻结策略快照，禁用来源必须完全退出 loading、检索、排序、引用、命中统计和回答上下文。

内部资料命中优先；内部无命中时才使用已开启的模拟官方开源资料或上游 Issue/PR；所有资料无命中时，按通用知识降级开关输出低置信度一般建议或信息不足提示。增加内置模拟截图和模拟日志的添加、预览、删除、随消息发送，以及只读诊断命令复制。

所有回答必须包含依据来源、已启用来源、实际命中来源、置信度、人工介入建议、待补充信息和非最终根因声明。所有示例标注模拟数据，链接使用 `example.com`，SLA 提交和建议适用性必须由人工确认。同步目标 API、检索过滤、安全审计、观测指标和专项验收设计，但不得宣称当前已实现真实能力。

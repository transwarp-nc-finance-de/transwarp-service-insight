# 安全边界

Status: ACTIVE  
Owner: 安全负责人  
Last reviewed: 2026-07-15
Source of truth for: 当前安全硬边界与未来接入门禁

## 当前边界

当前仅允许模拟、公开或已脱敏输入。服务只接本地 Compose PostgreSQL 保存版本化模拟身份、模拟目录与 AuthSession，不接真实身份系统、知识库、企业共享/生产数据库、模型、AIOps、ITSM 或生产环境；四个 `local-identity-v1` 模拟主体与策略 `mock-policy-v1` 不代表真实鉴权。审计和日志不得记录标题、描述、追问、反馈正文、Token、密钥或附件正文。

所有输出必须包含依据来源、置信度、人工介入建议和待补充信息，并标识 `模拟数据`。输出不得成为最终根因、最终方案或正式复盘结论；失败、低置信度和反馈失败不得阻断人工继续提交。

## 未来权限原则（DRAFT）

权限顺序固定为：身份 → 策略快照 → 可访问范围 → 候选过滤 → 检索 → 生成 → 展示。禁止先检索全部资料再做回答后过滤。文档级权限由 Chunk 继承，API 读取仍需再次校验。

知识文档和工具返回均视为不可信数据，不得修改系统规则。只允许白名单只读工具，禁止任意 URL 抓取、执行文档内命令、直接访问生产数据库或自动执行生产变更。

## 人工门禁

真实数据、真实 SSO/权限、数据保留周期、外部模型、外部系统、生产部署、自动操作和已发布 API 破坏性变化均需明确人工确认。

## 一期已确认本地文件边界

原始知识文件只允许写入 Compose 管理的挂载目录，并通过不透明文件 ID 与受控存储 Adapter 访问。数据库、业务 API、审计事件和日志不得记录或返回宿主机绝对路径。文件内容仍仅限模拟、公开或脱敏数据；该目标边界尚未实现。

上传在持久化前执行原始字节数校验：Markdown/TXT 最大 5 MB，文本型 PDF 最大 20 MB。超限返回不含内部路径或堆栈的安全错误，任何本地角色均不得绕过。

## 一期已确认本地身份边界

一期只允许预置 `模拟数据` 用户通过本地 Mock 登录/切换建立服务端 Session/Cookie。该 AuthSession 基础切片已实现；后端提供角色与产品线范围的 `IdentityContext`，后续受保护业务路径仍必须各自执行授权，前端隐藏按钮不构成授权控制。不实现注册、密码找回、真实组织同步、JWT 或正式 SSO。`IdentityContextPort` 隔离本地身份与未来宿主身份。

一期 RBAC 固定为 `PRECHECK_USER`、`KNOWLEDGE_EDITOR`、`KNOWLEDGE_REVIEWER`、`ADMIN`。管理员权限不构成业务职责绕过：不得批准自己提交的知识版本，也不得替用户确认继续提交。

一期数据权限为角色与预置产品线范围的交集，Chunk 继承文档范围。过滤必须先于全文和向量召回；无权限资料不得进入候选、引用、生成上下文或非必要审计摘要。组织树、任意文档 ACL 和用户组治理推迟到后续阶段。

## 一期已确认审计边界

审计只保存结构化元数据和稳定引用 ID；业务正文保留在原领域对象中，需要查看时重新授权。审计明确禁止保存明文密钥、Token、宿主机路径、附件内容、未脱敏真实数据和模型内部思维过程。

一期问题附件仅使用 `模拟数据` 的不透明 ID、文件名、媒体类型和大小，不读取、上传、解析或保存内容。附件元数据不得携带宿主路径、可直达内容的 URL 或长期凭据。知识治理原始文件适用独立的 `FileStoragePort`，两类文件不得混淆。

引用只通过受控 Evidence ID 访问不可变知识版本与 Chunk，不返回本地绝对路径或直接文件 URL。每次打开引用都重新校验当前身份与产品线范围；历史 Run 中曾经可见不代表当前仍有权访问。

本地数据重置只允许 `ADMIN` 对明确标识的本地环境执行，必须二次确认并审计。该能力只重建版本化 `模拟数据`，不是备份、恢复、灾备或生产删除流程。

一期本地数据不自动过期，也不提供用户逐条删除，统一保留至受控重置。该规则仅适用于模拟、公开或脱敏数据；真实数据接入前必须重新确认保留期限、删除一致性和审计保留例外。

Embedding 模型供应链固定模型 ID、revision、必要文件清单和 SHA-256。模型只允许在镜像构建阶段从受控来源下载，运行时禁止联网；不得执行远程自定义代码，也不得把未经复核的模型制品提交仓库或装入默认镜像。

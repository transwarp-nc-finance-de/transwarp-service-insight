## 当前边界

当前仅允许模拟、公开或已脱敏输入。服务不接真实身份系统、知识库、数据库、模型、AIOps、ITSM 或生产环境；模拟主体 `mock-user` 和策略 `mock-policy-v1` 不代表真实鉴权。进程内审计不得记录标题、描述、追问、反馈正文、Token、密钥或附件正文。

所有输出必须包含依据来源、置信度、人工介入建议和待补充信息，并标识 `模拟数据`。输出不得成为最终根因、最终方案或正式复盘结论；失败、低置信度和反馈失败不得阻断人工继续提交。

## 未来权限原则（DRAFT）

权限顺序固定为：身份 → 策略快照 → 可访问范围 → 候选过滤 → 检索 → 生成 → 展示。禁止先检索全部资料再做回答后过滤。文档级权限由 Chunk 继承，API 读取仍需再次校验。

知识文档和工具返回均视为不可信数据，不得修改系统规则。只允许白名单只读工具，禁止任意 URL 抓取、执行文档内命令、直接访问生产数据库或自动执行生产变更。

## 人工门禁

真实数据、真实 SSO/权限、数据保留周期、外部模型、外部系统、生产部署、自动操作和已发布 API 破坏性变化均需明确人工确认。

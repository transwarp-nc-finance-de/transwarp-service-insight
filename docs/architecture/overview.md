# 系统上下文

## 目的

本文档描述 `SLA 智能预诊助手` 与用户、ITSM、AIOps、知识库、大模型服务和指标系统之间的关系。

## 适用范围

适用于上下文边界评审、集成方案讨论和后续接口设计。当前 Demo 只模拟这些关系，不接真实系统。

## 正文

| 外部对象 | 目标关系 | 当前状态 |
| --- | --- | --- |
| 内部员工 | 在提交 SLA 前触发预诊、查看建议、人工确认后继续操作 | 纯前端模拟 |
| ITSM / AIOps | 后续作为工单创建、历史 SLA 查询和提交承载系统 | 未接入 |
| 知识库 | 后续提供授权、脱敏、审核通过的知识资料 | 未接入 |
| 大模型服务 | 后续基于检索结果生成结构化辅助建议 | 未接入 |
| 指标系统 | 后续接收埋点事件并形成效果度量 | 未接入 |
| 安全与审计 | 后续负责权限、脱敏、日志和追溯 | 未接入 |

当前阶段所有页面资料均为 `模拟数据`，不代表真实客户、真实 SLA、真实 Wiki、真实日志或真实内部链接。

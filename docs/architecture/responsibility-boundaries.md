# AIOps 与 Service Insight 职责边界

状态：ACTIVE。当前全部联调内容为 `模拟数据`。

| 能力 | AIOps | Service Insight |
| --- | --- | --- |
| SLA 表单、枚举和原有校验 | 权威来源 | 仅接收快照 |
| SLA 编号与最终提交 | 负责，人工确认 | 不创建、不阻断 |
| 信息完整度、辅助建议与引用 | 展示入口 | 负责 |
| 最终根因、方案与复盘 | SLA 处理人员认定 | 不负责 |
| 反馈、审计和知识治理 | 配合采集 | 负责分析 |

生产路径为：AIOps 表单 → 智能预诊 → 传入未提交快照 → Service Insight 返回结构化辅助结果 → 用户补充、参考、部分采纳或忽略 → 人工确认 → AIOps 原流程提交。任何状态均保留 `CONTINUE_SUBMISSION`。

# 模板

## 目的

本文档提供 `Transwarp Service Insight` 一阶段 `SLA 预诊助手` 的 SLA 提交、预诊、信息补全、相似案例、解决方案建议、SLA 草稿、复盘报告和知识库沉淀模板。

## 适用范围

适用于 Demo、需求评审和后续原型设计。模板中的示例内容均为模拟数据，正式使用前必须由人工审核。

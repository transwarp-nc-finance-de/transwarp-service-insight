# 当前范围：技术 MVP

## 目的

提供当前阶段唯一的短版范围入口，统一说明已完成内容、M1 目标和边界。

## 适用范围

适用于技术 MVP 的开发、评审、演示和验收。

## 正文

第一阶段纯前端 `SLA 智能预诊助手` Demo 已于 2026/07/10 关闭，历史原型归档在 `docs/archive/demo-v1/`。当前为技术 MVP 开发，M1 交付可本地运行的 Vue 前端与 Spring Boot Mock API 闭环。

M1 包含 SLA 表单、`POST /api/v1/precheck`、确定性模拟预诊、结构化错误、结果面板、人工继续提交模拟、前后端测试和启动文档。所有参考资料和结果均为 `模拟数据`；不接数据库、真实 RAG、真实模型、AIOps、ITSM、Wiki、历史 SLA 或生产数据。

M1 非目标包括鉴权、持久化、文件上传、真实提交、生产部署、自动闭单、最终根因判断和知识库沉淀。建议仅供人工参考，失败或信息不足不得阻断人工继续提交。

旧 PRD、架构、RAG、安全、API 和门禁文档继续作为历史基线或后续目标架构参考，不代表其中能力已实现。`docs/00-project/next-stage-decision-gate.md` 等材料用于真实试点阶段参考，不阻塞本地 Mock 技术 MVP。

数据仅限模拟、公开或已脱敏内容。涉及真实数据、外部服务、破坏性操作、公共接口兼容或重大业务歧义时必须人工确认。

# 安全与护栏设计

## 目的

本文档定义 SLA 智能预诊后续目标架构中的安全、合规、幻觉拦截和人工审核护栏。

## 适用范围

适用于安全评审、LLM 评审、RAG 评审和集成评审。当前阶段不接真实敏感数据。

## 正文

安全设计包括：数据不出域、权限过滤、脱敏、传输与存储加密、Prompt 注入防御、指令与用户输入隔离、生成前拦截、生成中约束、生成后引用对齐、实体一致性检查、敏感信息过滤、高风险结果降级、审计日志和人工审核。

模型输出只作为辅助建议，不得作为最终根因、最终处理结论或正式复盘结论。系统不得绕过人工审核，不得输出未脱敏敏感信息，不得根据用户注入指令改变系统规则。

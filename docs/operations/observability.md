## 当前证据

当前仅有进程健康接口、结构化 API 错误 `traceId`、进程内脱敏业务事件、容器健康检查和自动化测试。内存事件重启即清空，不是正式审计或指标平台。

## 目标指标

- API 成功率与 P50/P95/P99 延迟；
- 预诊、检索、生成和外部调用耗时、失败率与降级率；
- 知识解析、审核、索引任务状态和积压；
- 策略、规则、模型、Prompt、索引版本及单请求成本；
- 反馈、引用点击、信息补充、继续提交和知识缺口事件；
- 越权、敏感信息、Prompt 注入和工具策略违规事件。

## 日志与审计限制

禁止记录完整 Token、密钥、未脱敏正文、附件内容、模型内部思维过程或无权限链接。观测平台访问也必须遵循最小权限、保留期限和删除策略。

真实指标后端、日志平台、告警阈值、值班责任及保留周期尚未批准。
